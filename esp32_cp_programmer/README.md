# CP2102_iotmaker

## Features

CP2102 IOTMAKERVN:
- Size: 22x30.5 mm.
- Intergrate micro USB.
- Input Voltage: 5v (Vusb).
- Open hardware with CC-BY-SA license.
- Design with KiCad.
